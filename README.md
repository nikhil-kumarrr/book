# Book Recommendation System ‚Äî Streamlit App

A fast and simple Book Recommendation System built using Streamlit, Python, and collaborative filtering.
It uses precomputed similarity scores, a pivot table (pt), and a cleaned books dataset from Kaggle to deliver accurate book recommendations.

## Features
### Popular Books :
* Displays the most popular books from the dataset
* Shows title, author, rating, and book cover
* Clean UI with card-based layout

### Book Recommendations :
* Select a book from the dropdown
* Recommends 5 similar books
* Uses cosine similarity on user-item matrix
* Shows cover image, title, and author
  
### UI Enhancements :
* Full black background
* White clean book cards
* Hover animations
*Responsive Grid Layout

### Tech Stack :
* Python
* Streamlit
* Pandas
* NumPy
* Scikit-Learn
* Pickle
